Skills

Python frameworks

Django and Flask

Python has awesome libraries for data science. Therefore, an online web framework that supports a Python backend is a great choice for data driven web apps. Django in particular, provides a great way to handle databases and schema migrations without actually writing SQL code.
The main drawback... they're slow and heavy.

Javascript web frameworks

NodeJS + ExpressJS

A lot faster than python backends. Not as readable, though. Node and Express are a great combo to design responsive and UX intensive sites.
The main drawback is the lack of ORMs and nice SQL database integrations. If a database will be changing a lot when using these pair, mongo might be a better alternative than SQL.

Cloud infrastructure

AWS: Amazon Web Services

AWS is a great tool to develop cloud infrastructure. You can start on-demand servers, database servers, lambda functions, message queues, ETL services, storage and static web hosting, and the list keeps growing. AWS provides great ways to consume data streams, store data, train machine learning models, and render results; without investing in hardware.
